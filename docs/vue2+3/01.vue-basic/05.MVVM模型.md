```
MVVM 模型
  1. M：模型 (Model) ：data 中的数据
  2. V：视图 (View) ：模板代码
  3. VM：视图模型 (ViewModel)：Vue 实例
观察发现：
  1.data 中所有的属性，最后都出现在了 vm 身上。
  2.vm 身上所有的属性，及 Vue 原型上所有属性，在 Vue 模板中都可以直接使用。
```
